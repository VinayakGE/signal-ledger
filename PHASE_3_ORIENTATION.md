## Phase 3 Success Criterion

Not: "Close questions"

But: "Know exactly why questions remain open"

---

## The Distinction

### Closed
Question has been answered.
Evidence supports a conclusion.
Investigation can move forward.

### Unresolved
Question remains, but framework exists to address it.
Investigation has defined what would resolve it.
Clear falsification pathway exists.

### Currently Unresolvable
Question is real, but available evidence cannot address it.
Repository knows the evidence class needed (and it's not available).
Future cases or future evidence types may allow resolution.

---

## Why This Is A Higher Standard

A weak research program closes questions prematurely.

A stronger one distinguishes between these three states and records which state each question is in.

---

## Practical Application for Phase 3

### When a question emerges, ask:
1. Is it closed? (Evidence supports answer)
2. Is it unresolved? (Could be resolved; pathway exists)
3. Is it currently unresolvable? (Evidence class unavailable; but identifiable)

Record which state.

Do not try to force questions into "closed" when they belong in "unresolvable."
