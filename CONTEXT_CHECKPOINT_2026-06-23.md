## Investigation Discipline Summary

**What's Different About This Phase 2 Work:**

1. **Pre-registration:** Falsification conditions frozen BEFORE evidence review (not after)
2. **Wound-seeking:** Investigators look for what would kill objects (not patterns confirming them)
3. **Reality-based:** Evaluation uses observable facts (not object's own language)
4. **Honest scoring:** Results reported regardless of predictions (no rationalization)
5. **Multi-axis:** Objects evaluated on Strength + Survival + Generativity (not just survival)

**Why this prevents error:**

- Pre-registration blocks post-hoc rationalization
- Wound-seeking blocks confirmation bias
- Reality-based blocks interpretive flexibility
- Honest scoring blocks self-protection
- Multi-axis blocks single-dimension overconfidence
