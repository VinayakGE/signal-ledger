## Critical Decisions Made

**Why WeWork as adversarial test?**

1. Different organizational type (private founder-led vs. institutional)
2. More recent failure (better documentation expected)
3. Abundant public record (better sourcing than Kodak/Nokia)
4. Founder narrative (sharper Zone A/B boundary than hardware product companies)
5. High leverage for M1 test (does explanatory object survive?)

**Why pre-registration?**

1. Prevents post-hoc rationalization
2. Separates prediction from observation
3. Makes falsification conditions explicit
4. Guards against confirmation bias

**Why wound-seeking posture?**

1. Objects must earn survival through surviving attack
2. Finding patterns that confirm predictions is low-value
3. Finding evidence that could wound is high-value
4. Learning happens when predictions fail
