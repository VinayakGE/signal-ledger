## WeWork Unique Factors

**Why Class C might be MORE accessible:**
- Private company (less privacy norms than institutions)
- Recent failure (more documentation exists)
- Adam Neumann public figure (more interviews)
- Extensive media investigation (more sources)

**Why Class C might be LESS accessible:**
- Private company (no regulatory disclosure requirements)
- Neumann may actively withhold (reputation concerns)
- Records may not have been preserved (young company)
- Post-failure, selective documentation
