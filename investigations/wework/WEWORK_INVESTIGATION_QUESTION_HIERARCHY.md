**Purpose:** Establish which questions are primary and which are secondary

## Question Hierarchy (In Priority Order)

### TIER 1: Investigation Questions (PRIMARY)

These are about WeWork, not about Phase 2 objects.

#### Question 1.1: What is most surprising about WeWork's failure?

**What it asks:**
- What does WeWork reveal that contradicts existing organizational failure theory?
- What pattern appears in WeWork that wasn't predicted by any framework?
- What makes WeWork's failure mode distinct from Kodak/BlackBerry/Nokia/GE?

**Why it's primary:**
- Drives genuine discovery (not confirmation)
- May reveal phenomena Phase 2 objects weren't designed to detect
- Highest learning value if WeWork surprises us

**Success condition:**
- Investigation is guided by "What's unexpected?" not "What confirms objects?"
- WeWork findings are allowed to diverge from Phase 2 predictions
- Most surprising findings are highlighted, not buried

#### Question 1.2: What can we observe about Neumann's decision-making?

**What it asks:**
- What strategic choices are documented?
- What evidence exists about motivation for those choices?
- What remains opaque despite extensive public record?

**Why it's primary:**
- Establishes observational foundation (independent of objects)
- Creates Zone B mapping without prejudging Zone A
- Allows classification of evidence by type (A/B/C) before interpretation

**Success condition:**
- Strategic choices inventory is complete and objective
- Evidence is classified by type and date (not by whether it supports objects)
- Motivation opacity is assessed before consulting objects

#### Question 1.3: What does WeWork's evidence landscape reveal about investigation limits?

**What it asks:**
- Which classes of evidence are systematically available or unavailable?
- Does founder-led private company show different evidence distribution than institutions?
- What questions about organizational failure remain unanswerable despite public record?

**Why it's primary:**
- Directly addresses Phase 2's core question (knowledge limits)
- May generalize beyond WeWork to other founder-led cases
- Contributes to evolving understanding of investigation methodology

**Success condition:**
- Evidence class distribution is documented (Classes A, B, C availability)
- Compared to institutional cases (Kodak/Nokia/GE/BlackBerry)
- Boundary questions are identified (what remains unknowable)

### TIER 2: Object Testing Questions (SECONDARY)

These are about Phase 2 objects in light of WeWork findings.

#### Question 2.1: Does Evidence Class Taxonomy hold in new domain?

**What it asks:**
- Is Class C (contemporaneous motivation) scarce in founder-led private companies?
- Or is scarcity institutional-specific?

**Status:** Secondary (depends on Tier 1 findings)

**Falsification condition:** Class C evidence abundant (contemporaneous Neumann motivation documents)

#### Question 2.2: Does Zone A/B boundary apply to founder-led companies?

**What it asks:**
- Is founder motivation typically opaque (Zone A) even with public founder?
- Or is founder-led domain more transparent?

**Status:** Secondary (depends on Tier 1 findings)

**Falsification condition:** Founder motivation becomes transparent (private reasoning documented)

#### Question 2.3: Does PP#002 appear independently in WeWork?

**What it asks:**
- Without deliberate hunting, does observable/opaque pattern emerge naturally?
- Is WeWork's pattern similar to Kodak/BlackBerry/GE?

**Status:** Secondary (depends on Tier 1 findings)

**Falsification condition:** Contemporaneous transparency in motivation (Class C documents explaining choices)

---

### TIER 3: Meta-Level Questions (TERTIARY)

These are about what Phase 2 itself is discovering.

#### Question 3.1: Is M1 supported or wounded?

**What it asks:**
- Does WeWork produce durable explanations or primarily boundaries?
- If durable explanation found, does it survive like boundaries did?
- Is M1's prediction (explanations die faster) holding?

**Status:** Tertiary (emerges from Tier 2 results)

**Note:** Durable explanation would be HIGH-VALUE outcome (wounds M1; teaches about knowledge types)

#### Question 3.2: What is the repository discovering about investigation?

**What it asks:**
- Is Phase 2's output primarily investigation methodology (boundaries/instruments)?
- Or mix of methodology + organizational explanations?
- Which kinds of knowledge survive investigation?

**Status:** Tertiary (emerges from Tier 1+2 synthesis)

**Note:** This is the deepest question Phase 2 is exploring

## How Question Hierarchy Guides Investigation

### During Evidence Collection

**Tier 1 priority:** Observe WeWork's choices and evidence landscape without prejudging objects

**Avoid:**
- ✗ Searching only for evidence that confirms/denies objects
- ✗ Interpreting evidence through object lens
- ✗ Dismissing findings unrelated to Phase 2

**Do:**
- ✓ Classify all evidence (A/B/C) independently
- ✓ Document strategic choices objectively
- ✓ Note surprising or unexpected patterns first

### During Analysis

**Step 1:** Analyze Tier 1 questions (What does WeWork tell us about investigation?)

**Step 2:** Use Tier 1 findings to analyze Tier 2 questions (How do objects perform?)

**Step 3:** Synthesize Tier 1+2 to address Tier 3 questions (What is repository discovering?)

**Do not:**
- ✗ Start with "Do objects survive?" (that's Tier 2)
- ✗ Jump to "Is M1 correct?" (that's Tier 3)
- ✗ Filter observations through object lens

### During Synthesis

**Leading finding:** Most surprising discovery (from Tier 1)

**Supporting findings:** How objects performed (from Tier 2)

**Meta-implication:** What this teaches about investigation (from Tier 3)

## Operationalizing the Hierarchy

### Question 1.1 Implementation: "What's Most Surprising?"

**Before concluding investigation, explicitly answer:**

1. What did WeWork reveal that contradicted expectations?
   - Not Phase 2 objects' expectations
   - Expectations from Phase 1 (organizational failure patterns)

2. What pattern appeared that wasn't explicitly predicted?
   - By Phase 2 objects
   - By Phase 1 hypotheses
   - By any external theory

3. If we had to highlight ONE finding WeWork discovered, what would it be?
   - Must be something genuinely surprising
   - Can be unrelated to Phase 2 objects
   - Will guide Phase 3 direction

**This must be answered first, before object-level findings.**

### Question 1.2 Implementation: "What's Observable About Decisions?"

**Inventory WeWork's documented strategic choices:**

1. Each choice (capital escalation, IPO timing, model conviction)
2. Observable evidence (what's documented)
3. Evidence classes (A/B/C availability)
4. Motivation assessment (transparent or opaque)

**All completed independently of whether findings support/wound objects.**

### Question 1.3 Implementation: "What Does Evidence Landscape Reveal?"

**Explicitly map:**

| Evidence Type | WeWork | Kodak | Nokia | GE | BlackBerry | Pattern |
|---------------|--------|-------|-------|-----|-----------|---------|
| Class A | Abundant | Abundant | Moderate | Abundant | Abundant | Universal |
| Class B | Moderate | Moderate | Limited | Blocked | Moderate | Varies |
| Class C | [TBD] | Scarce | Scarce | Blocked | Scarce | Institutional? |

**Conclusions drawn from pattern, not from object predictions.**

## Success Criteria (Revised)

### Investigation Success

**Tier 1 questions fully answered:**
- ✓ Most surprising finding identified and documented
- ✓ Strategic choices inventory complete
- ✓ Evidence landscape mapped
- ✓ No forced interpretation through object lens

### Object Testing Success

**Tier 2 questions assessed:**
- ✓ Each object scored independently (Survived/Wounded/Refuted)
- ✓ No rationalization of contradictory evidence
- ✓ Wounds documented explicitly

### Meta-Level Learning

**Tier 3 synthesis complete:**
- ✓ M1 status updated based on actual findings
- ✓ Repository's methodological discovery documented
- ✓ Phase 3 direction informed by what was learned

## Critical Redirects (If Investigation Drifts)

### If you find yourself asking...

**"Will this wound PP#002?"**
→ Stop. First ask: "Is this the most important finding?"

**"Does Zone A/B boundary hold?"**
→ First answer: "What's most surprising about Zone A/B in WeWork?"

**"Did objects survive?"**
→ First answer: "What did WeWork teach us about investigation?"

**"Is our methodology sound?"**
→ First answer: "What did WeWork discover that we didn't expect?"
