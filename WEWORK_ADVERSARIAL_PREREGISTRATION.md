## Object #1: Evidence Class Taxonomy

### Prediction (Made During Compression Pass)

Class C evidence (contemporaneous motivation, decision rationale at time T) will be **systematically inaccessible** in WeWork because:
1. WeWork is private company (no regulatory disclosure requirements like GE, Kodak)
2. WeWork's records may not have been preserved (unlike 100-year-old institutions)
3. Even if preserved, founder may actively decline to disclose (like Lazaridis, Kodak executives)

### Falsification Condition: Evidence Taxonomy Weakens If

**Class C evidence is readily accessible** in WeWork — specifically:

- Contemporaneous board memos from 2015-2019 explicitly documenting Neumann's strategic rationale
- Internal emails or strategy notes written before IPO failure (pre-September 2019)
- Decision rationale documentation from Neumann, board members, or investors showing contemporaneous reasoning

**Example of Class C that would wound:**
- Neumann's 2018 board memo: "Why I believe the unit-economics concern is misplaced and growth will eventually justify valuation"
- Investor strategy notes 2015-2018 showing explicit reasoning for capital allocation decisions

### Why This Matters

If Class C evidence is abundant in WeWork, the taxonomy's prediction fails. The taxonomy predicted Class C rarity; abundance would wound it.

### Search Protocol

1. Search for: "WeWork Neumann board memo," "WeWork internal strategy," "WeWork decision rationale documents"
2. Classify any contemporaneous decision documents found as Class C
3. Count available Class C evidence
4. Compare to availability in Kodak, Nokia, GE, BlackBerry

### Outcome Scoring

- **Taxonomy survives:** Class C remains scarce (consistent with prediction)
- **Taxonomy wounded:** Class C found in abundance (prediction failed; taxonomy loses strength)

## Object #2: Zone A/B Boundary

### Prediction (Made During Compression Pass)

Zone A/B boundary maps to organizational structure generally, not just institutional complexity. Specifically:

- **Zone B (Observable):** Founder's public narrative (Adam Neumann's statements, investor pitches, media interviews)
- **Zone A (Opaque):** Founder's private reasoning (why Neumann believed in the model; whether conviction or performance theater)

### Falsification Condition: Zone A/B Weakens If

**Founder's private reasoning is fully transparent** in WeWork — specifically:

- Neumann's private journals, personal papers, or therapeutic records showing unfiltered thinking
- Direct access to Neumann explaining his beliefs (not post-hoc rationalization)
- Board members documenting Neumann's expressed beliefs in real time (not retrospective interpretation)

**Example of evidence that would wound:**
- Neumann's personal diary entry (2017): "I genuinely believe this model will work because..." (contemporaneous conviction visible)
- Confidential investor communication where Neumann explains reasoning before outcome known

### Why This Matters

If Zone A becomes transparent, the boundary doesn't hold. The observation was that motivation remained opaque; abundant transparency would refute it.

### Search Protocol

1. Search for: "Neumann personal papers," "Neumann diary," "Neumann private correspondence," "Neumann journals"
2. Look for contemporaneous first-person expressions of belief (not retrospective)
3. Classify evidence as Class B (retrospective) vs. Class C (contemporaneous)
4. Assess whether Zone A remains opaque or becomes transparent

### Outcome Scoring

- **Zone A/B survives:** Founder's motivation remains private; observable narrative differs from opaque conviction
- **Zone A/B wounded:** Founder's private reasoning is documented; transparent access to real-time beliefs

## Object #3: PP#002 (Observable Selection, Opaque Motivation)

### Prediction (Made During Compression Pass)

PP#002 pattern will appear in WeWork **independently and naturally**, without investigator hunting for it:

- **Observable:** Neumann made documented strategic choices (capital raises, IPO timing, governance structure)
- **Selection visible:** Decisions between documented alternatives (growth-at-any-cost vs. unit-economics focus; IPO now vs. later)
- **Motivation opaque:** Why those specific choices were made remains unrecoverable from public evidence

### Falsification Condition: PP#002 Weakens If

**Motivation becomes transparent** through any of these:

**Strongest wound (complete refutation):**
- Neumann's contemporaneous writings explain strategic rationale before IPO failure (Class C)
- Board minutes document Neumann's stated reasoning for growth-focused capital allocation
- Investor communications show Neumann explaining his conviction in the model

**Moderate wound (pattern doesn't hold):**
- Neumann discusses his motivations in interviews BEFORE IPO failure (not post-hoc)
- Internal strategy documents show explicit reasoning that was written in real time

**Weak wound (insufficient to refute):**
- Post-IPO-failure interviews where Neumann explains his thinking (outcome-contaminated; Class B)
- Board members' retrospective accounts of what they think Neumann believed

### Critical Distinction for This Object

**Hunt independently; don't retrofit:**

The pattern should be:
1. Observe WeWork's strategic choices (capital structure, IPO decision, governance)
2. Ask "what drove these choices?" naturally
3. If motivation remains opaque → PP#002 survives
4. If motivation becomes transparent → PP#002 wounded

**NOT:**

1. Ask "does WeWork show Observable/Opaque pattern?"
2. Search for evidence to fit the pattern
3. Call it "found" if motivation is somewhat unclear

The first is validation. The second is confirmation bias.

### Search Protocol

1. Inventory WeWork's documented strategic choices (capital rounds, timing decisions, governance structure)
2. For each choice, ask: what drove this?
3. Look for evidence of motivation (explicit reasoning, decision rationale)
4. Classify evidence by class (A/B/C)
5. Ask: does motivation remain opaque?

### Outcome Scoring

- **PP#002 survives:** Observable choices documented; motivation remains opaque despite extensive public record
- **PP#002 strengthened:** Observable choices documented; motivation opaque even with founder interview attempts
- **PP#002 wounded:** Observable choices documented; contemporaneous reasoning found in accessible sources (Class C evidence)
- **PP#002 refuted:** Motivation becomes fully transparent; founder's private reasoning documented

## Investigation Scope

**Research question:** 
Given information available before WeWork's IPO failure (September 2019), what observable strategic choices can we identify, and what proportion of the motivational reasoning for those choices remains opaque despite founder and investor access?

**Outcome known:** Yes (Backtest) — IPO failed; company collapsed; extensive post-failure documentation exists

**Information boundary:** Evidence available or claimed to be available before September 2019

**Post-hoc evidence:** Allowed for Class B classification and outcome-contamination marking. Not allowed for claiming contemporaneous belief without Class C validation.

- Evidence Class Taxonomy: Class C remains scarce ✓
- Zone A/B Boundary: Founder motivation opaque ✓
- PP#002: Observable choices, opaque motivation ✓

**Implication:** Objects are real, not artifacts. Phase 2 instruments are validated. Confidence in portfolio-level patterns increases substantially.

- Zone A/B Boundary: Founder motivation transparent (private journals, clear reasoning)

**Implication:** Founder-led private companies may not show Zone A opacity. Organizational complexity creates opacity differently in institutions. Objects partially domain-specific.

**Implication:** Observable/Opaque boundary doesn't hold in all cases. Founder-led companies may allow more visibility into motivation than institutions. Pattern is weaker than portfolio-level artifact status.

**Next step:** Downgrade PP#002 to "context-dependent." Keep other objects.

### Scenario E: Multiple Objects Wounded

**Implication:** Instruments are overfitted to institutional cases. Not yet general.

**Next step:** Return to Kodak/BlackBerry/GE. Investigate why those cases produced patterns. Revise instruments for institutional vs. founder-led domain differences.

## Guard Rails

- ✗ Decide in advance that "WeWork will definitely show the pattern"

- ✗ Rationalize why unexpected evidence "doesn't really count"
- ✗ Modify falsification conditions after seeing evidence
- ✗ Accept weaker wounds as sufficient
- ✗ Claim all three objects "survived" if one actually was wounded

### Do Do

- ✓ Let evidence guide observation
- ✓ Record when falsification conditions are approached
- ✓ Acknowledge wounds explicitly
- ✓ Mark outcome-contaminated evidence as Class B
- ✓ Score each object independently
- ✓ Report honest results, including wounds
