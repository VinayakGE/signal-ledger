## Repository Disciplines Established

### Methodological Discipline ✓

- Pre-registration of falsification conditions
- Separation of prediction from observation
- Wound-seeking posture (not pattern-matching)
- Evidence classification by type and date
- Guard rails against confirmation bias

### Epistemic Discipline ✓

- Strength vs. Survival orthogonal axes
- Durability ≠ Value distinction
- Lower-risk vs. higher-risk claims
- Inference warnings for presupposed answers
- Documentation of object confidence levels

### Investigative Discipline ✓

- Phase structure (Audit → Compression → Validation)
- Pre-registration blocks post-hoc rationalization
- Objects scored independently
- Honest reporting of wounds
- No special pleading for predictions

## Repository Innovation: The Pre-Registration Discipline

**What makes WeWork investigation different from earlier phases:**

Phase 1 question: "What pattern appears in this case?"
Phase 2 question: "What would kill our objects? Do we find it?"

WeWork applies Phase 2 discipline:
- Falsification conditions frozen before evidence review
- Prediction locked in (cannot modify based on findings)
- Search is wound-seeking, not pattern-matching
- Results are honest regardless of object survival

**This is the shift from "explanatory programs" to "testing programs."**
